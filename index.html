Projects

Sukoon - The Music App Sep 2023 - Nov 2023

Sukoon is a consumer facing music app allowing users to listen to music, read lyrics, review songs and create playlists.

Users can also register as Creator to upload songs and create albums which can be searched and used by normal users. Creators can also see how their creations are performing by using pre-defined metrics. CRUD operations on songs, playlists and albums are provided.

An admin persona is enabled for administrators of the application to create master data for the application and assess how songs and creators are performing. Application also features some salient KPIs shown as charts such as - users distribution, daily active users, number of songs created in last days. Administrators can also blacklist creators if their creations are not compliant with content policies. Admins can also see five highest rated songs and five most active creators.

This application is implemented using Python and the pages are rendered on server side. SQLite database is used to store data and media files (thumbnails, songs, etc.) are stored on file system.
